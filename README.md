# 尚品甄选项目

## 项目介绍

尚品甄选是一个B2C模式的电子商务平台，包含后台管理系统和前台用户系统。项目采用前后端分离开发模式，基于SpringBoot + SpringCloud微服务架构，集众多主流技术于一身，是一款最适合新手入门的企业级项目。

## 技术栈

| Name       | Version |
| ---------- | ------- |
| JDK        | 1.7     |
| SpringBoot | 3.0.5   |
| MySQL      | 8.0.30  |
| Redis      | 7.0.10  |

## 业务功能

用户登录、权限管理、商品管理、商品分类、商品规格管理、订单统计
